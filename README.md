# EvE

EvE is a prototype app developed with Ionic 2. It's a shopping list app which supports users in making environmentally friendly choices when shopping for food. [Eaternity](https://eaternity.org/) has generously provided me with access to their database, which is used in this project. [View additional details for the project](https://luisaschleu.artstation.com/projects/eX28w)
